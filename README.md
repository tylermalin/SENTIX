# Sentix Network Landing Page

Industrial Sci-Fi landing page for Sentix Network ($SNTX) - The Currency of Computational Thought.

## Tech Stack

- **Next.js 14** (App Router)
- **TypeScript**
- **Tailwind CSS**
- **Framer Motion** (Animations)
- **Lucide React** (Icons)

## Getting Started

```bash
# Install dependencies
npm install

# Run development server
npm run dev

# Build for production
npm run build

# Start production server
npm start
```

Open [http://localhost:3000](http://localhost:3000) to view the site.

## Design System

- **Backgrounds:** Deep charcoal and slate black
- **Accents:** Electric Cyan (#00F0FF) and Deep Silver
- **Typography:** Inter (Body), JetBrains Mono (Technical)
- **Theme:** Industrial Sci-Fi / Bloomberg Terminal aesthetic
